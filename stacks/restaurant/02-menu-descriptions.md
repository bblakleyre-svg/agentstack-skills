## Purpose
Write menu copy that makes guests hungry, sets accurate expectations, highlights what makes each dish special, and reduces order anxiety — especially for new guests.

---

## When To Use
- Creating or updating a menu
- Adding seasonal specials
- Writing online menu copy (website, Google, delivery apps)
- Training staff on how to describe dishes

---

## Instructions

### Menu Description Structure
1. **Lead with the hero ingredient or technique** — what makes it special?
2. **Supporting ingredients** — 2–4 key elements, not every ingredient
3. **Flavor or texture note** — what will they experience?
4. **Preparation or origin note** (if distinctive) — house-made, wood-fired, local source
5. **Dietary tags** — [GF] [V] [VG] [DF] — keep in a consistent format

### Writing Rules
- Sensory words sell: crispy, silky, charred, bright, briny, herbaceous, buttery
- Limit each description to 20–35 words — menus aren't novels
- Avoid pretentious language unless the brand calls for it
- Never describe every ingredient — feature the 3 that matter most
- Price should never be justified in the description — the food does that

### Description Tiers by Style

**Casual/Bistro:**
> Pan-seared salmon, lemon-caper butter, whipped potatoes, haricots verts. Bright and satisfying. [GF]

**Mid-Range/Contemporary:**
> Wood-grilled salmon fillet with house-made lemon-caper beurre blanc, chive mashed potato, and crispy green beans. Light, clean, and deeply satisfying. [GF]

**Fine Dining:**
> Wild Pacific salmon, seared to a lacquered crust, served over silken pomme purée with a bright lemon-caper emulsion and garden haricots. [GF]

## Examples

**Appetizer:**
> Burrata & Heirloom Tomatoes — Fresh burrata, vine-ripened tomatoes, aged balsamic, torn basil, sea salt. Summer in a bowl. [V, GF]

**Cocktail:**
> The Garden Mule — House-infused cucumber vodka, fresh lime, ginger beer, mint. Clean, crisp, dangerously drinkable.

**Seasonal Special:**
> Butternut Squash Risotto — Arborio rice, roasted squash, sage brown butter, toasted pepitas, aged Parmigiano. Warming and rich. [V]

---

## Tone Guidelines
- Match the restaurant voice — a diner writes differently than a Michelin-starred tasting room
- One emotional word per description is enough — "comforting" or "bright" — don't over-adjective
- Staff should be able to describe every dish from memory — write for that purpose

---

## Trigger Handoffs
- Menu updated → brief all FOH staff before next service
- Guest asks about a dish → this skill is the source of truth for server training
- Dish consistently returned or questioned → review description accuracy
